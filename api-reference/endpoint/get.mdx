# Listar Assistentes

Retorna uma lista paginada de todos os assistentes de IA configurados na sua conta.

## Endpoint

```http
GET https://api.sofia.ai/v1/assistants
```

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token com sua API key
</ParamField>

## Query Parameters

<ParamField query="page" type="integer">
  Número da página (padrão: 1)
</ParamField>

<ParamField query="limit" type="integer">
  Itens por página (1-100, padrão: 20)
</ParamField>

<ParamField query="search" type="string">
  Busca por nome ou descrição do assistente
</ParamField>

<ParamField query="status" type="string">
  Filtrar por status: "active", "inactive", "training"
</ParamField>

<ParamField query="language" type="string">
  Filtrar por idioma (código ISO 639-1)
</ParamField>

<ParamField query="sort" type="string">
  Campo para ordenação: "name", "created_at", "updated_at" (padrão: "created_at")
</ParamField>

<ParamField query="order" type="string">
  Ordem: "asc" ou "desc" (padrão: "desc")
</ParamField>

## Busca e Filtros

### Busca por Texto
O parâmetro `search` busca nos campos:
- Nome do assistente
- Descrição do assistente
- Tags (se configuradas)

**Exemplo:**
```
GET /v1/assistants?search=vendas
```
Retorna assistentes que contenham "vendas" no nome ou descrição.

### Filtros Disponíveis

#### Por Status
```
GET /v1/assistants?status=active
```
- `active`: Assistentes ativos e prontos para uso
- `inactive`: Assistentes desativados
- `training`: Assistentes em treinamento (voz clonada)

#### Por Idioma
```
GET /v1/assistants?language=pt-BR
```
Filtra assistentes configurados para idioma específico.

#### Combinando Filtros
```
GET /v1/assistants?status=active&language=pt-BR&search=atendimento
```

### Ordenação

#### Campos Disponíveis
- `name`: Nome do assistente (A-Z)
- `created_at`: Data de criação
- `updated_at`: Data de atualização
- `calls_count`: Número de ligações

#### Exemplos
```
GET /v1/assistants?sort=name&order=asc
GET /v1/assistants?sort=calls_count&order=desc
```

## Paginação

### Navegação
```javascript
// Primeira página
GET /v1/assistants?page=1&limit=10

// Próxima página
GET /v1/assistants?page=2&limit=10

// Verificar se há mais páginas
if (response.pagination.has_next) {
  // Carregar próxima página
}
```

### Limites
- Mínimo: 1 item por página
- Máximo: 100 itens por página
- Padrão: 20 itens por página

## Casos de Uso

### Dashboard de Assistentes
```javascript

```

```javascript

```


## Performance

### Cache
- Resultados são cacheados por 5 minutos
- Use header `Cache-Control: no-cache` para forçar atualização

### Rate Limiting
- Máximo 100 requests por minuto
- Use paginação para grandes volumes
